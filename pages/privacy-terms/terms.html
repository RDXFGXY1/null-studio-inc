3. Service Description

Hinata is a modular Discord bot that provides security, moderation, logging, and utility features. Core capabilities include, but are not limited to:

  • Anti-nuke protection - Detecting and mitigating mass role/channel/permission changes, mass kicks/bans, and automated rollback/recovery
  • Moderation automation - Auto-bans, kick detection, role-management protection
  • Comprehensive logging - Message edits, deletions, member join/leave, role updates, voice state changes, audit log events
  • URL scanning and threat detection - VirusTotal integration for scanning suspicious links
  • Spam and scam detection - Pattern analysis and automated mitigation
  • Backup and recovery utilities - Channel/role snapshots and restore functions
  • Premium subscription features - Tiered functionality accessed via secure payments
