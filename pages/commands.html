Logging Setup

Configure comprehensive server logging for all events

10+ Commands
/setlogchannel messages
Free

Configure logging channels for message-related events like edits, deletions, and reactions.

Usage

/setlogchannel messages [type]

Example

/setlogchannel messages content_modifications

Types Available

  • content_modifications - Message edits and deletions
  • pinned_messages - Pin and unpin activities
  • bulk_deletions - Mass message deletions
  • reaction_changes - Reaction additions/removals
/setlogchannel members
Free

Configure logging for member activities including joins, leaves, bans, and role changes.

Usage

/setlogchannel members [type]

Example

/setlogchannel members join_leave

Types Available

  • join_leave - Member joins and leaves
  • bans_kicks - Ban and kick actions
  • timeouts - Timeout applications
  • profile_changes - Nickname, avatar changes
  • role_updates - Role additions/removals
/setlogchannel voice
Free

Configure logging for voice channel activities and member voice state changes.

Usage

/setlogchannel voice [type]

Example

/setlogchannel voice connections

Types Available

  • connections - Join/leave voice channels
  • state_changes - Mute, deafen, stream status
/reset_setup
Free

Reset all logging configurations and start fresh with setup.

Usage

/reset_setup

Security Commands

Advanced protection systems including anti-nuke, spam filtering, and URL scanning

12+ Commands
/antinuke_setup
Premium

Configure anti-nuke protection system to prevent server destruction attacks. Premium feature with advanced restoration capabilities.

Usage

/antinuke_setup [alert_channel] [log_channel]

Example

/antinuke_setup #security-alerts #security-logs

Protection Against

  • Mass channel deletions
  • Mass role deletions
  • Mass member kicks/bans
  • Permission escalations
  • Malicious bot invasions
/antinuke_whitelist
Premium

Manage anti-nuke whitelist for trusted users and roles who can bypass protection systems.

Usage

/antinuke_whitelist [add/remove] [user/role]

Example

/antinuke_whitelist add @TrustedAdmin
/restore_server
Premium

Comprehensive server restoration after nuke attacks. Multiple restoration types available with hierarchical recovery.

Usage

/restore_server [type] [hours_back] [confirm]

Example

/restore_server full 24 CONFIRM

Restoration Types

  • full - Complete server restoration (requires CONFIRM)
  • channels - Text & voice channels only
  • recent - Last 10 deleted channels
  • emergency - Fast 6-hour recovery
