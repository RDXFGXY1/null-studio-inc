Troubleshooting

Common issues and solutions

Bot Not Responding

If NullTracker isn't responding to commands:

  • Check bot permissions
  • Ensure bot is online
  • Verify command syntax
  • Try in different channel
Solution: /ping to test connectivity

Commands Not Working

If slash commands aren't appearing:

  • Re-invite bot with updated permissions
  • Wait for Discord sync (up to 1 hour)
  • Check application.commands scope
  • Refresh Discord client
Solution: Use /help to see available commands

Anti-Nuke False Positives

If legitimate actions are being blocked:

  • Add trusted users to whitelist
  • Adjust protection thresholds
  • Review security logs
  • Configure role permissions
Solution: /whitelist add @user
