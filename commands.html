Command Reference

Complete guide to all NullTracker commands and features

Core Commands

/help

Free

Get help information for commands

Usage: /help [command]
Example: /help setlogchannel

/info

Free

View current server logging configuration

Usage: /info

/ping

Free

Check bot latency and response time

Usage: /ping

Logging Setup

/setlogchannel

Free

Configure logging channels for different event types

Usage: /setlogchannel [category] [type]
Categories:
  • messages - Message edits, deletions, reactions
  • voice - Voice channel activities
  • members - Member joins, leaves, role changes
  • roles - Role creation, deletion, updates
  • server - Server settings changes
  • moderators - Moderation actions
  • channels - Channel management
  • integration - Webhook and bot activities
Example: /setlogchannel messages content_modifications

/reset_setup

Free

Reset all logging configurations

Usage: /reset_setup

/reset_channel

Free

Reset specific log type configuration

Usage: /reset_channel [type]

Security Commands

/antinuke_setup

Premium

Configure anti-nuke protection system

Usage: /antinuke_setup [alert_channel] [log_channel]
Example: /antinuke_setup #security-alerts #security-logs

/antinuke_whitelist

Premium

Manage anti-nuke whitelist for trusted users/roles

Usage: /antinuke_whitelist [add/remove] [user/role]
Example: /antinuke_whitelist add @TrustedAdmin

/url_scanner_setup

Free

Configure URL scanner for malicious link detection

Usage: /url_scanner_setup [alert_channel]

/url_whitelist

Free

Manage trusted domains whitelist

Usage: /url_whitelist [add/remove] [domain]
Example: /url_whitelist add discord.com

/spam_setup

Free

Configure spam filter protection

Usage: /spam_setup

Server Restoration (Premium)

/restore_server

Premium

Comprehensive server restoration after nuke attacks

Usage: /restore_server [type] [hours_back] [confirm]
Restoration Types:
  • full - Complete server restoration (requires CONFIRM)
  • channels - Text & voice channels only
  • recent - Last 10 deleted channels
  • emergency - Fast 6-hour recovery
Examples:
/restore_server full 24 CONFIRM
/restore_server emergency

/restore_status

Premium

Check server restoration status and available backups

Usage: /restore_status

/recover_channels

Premium

Browse list of deleted channels that can be recovered

Usage: /recover_channels

/recover_channel

Premium

Recover a specific deleted channel by backup ID

Usage: /recover_channel [backup_id]
Example: /recover_channel 1234
